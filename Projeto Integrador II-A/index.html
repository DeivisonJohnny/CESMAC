Nas duas primeira linhas importamos duas biblioteca do python. random é usada em nosso código para escolher uma palavra de forma aleatória a partir de uma lista(palavras). Biblioteca OS é usada mais á frente para limpar o terminal após os a digitação de alguma letra

No próximo bloco definimos uma função (escolha_palavra), que escolhe e armazena em uma variável(palavra_escolhida) uma palavra aleatória da lista(palavras) e retorna a palavra escolhida(palavra_escolhida).

No próximo bloco a definimos a função(exibir_forca) com o parâmetro erros(exibir_forca(erros)) que imprimirar na tela o que esta dentro de uma lista(forca) a partir do números de tentativas que será passado pelo parâmetro(erros) como o índice desta lista

No próximo block definimos a nossa função principal, função(jogar), que armazena o que a função(escolha_palavra) retornar em uma variavél(palavra). A variável(palavra_mascarada) irá armazenar uma string que contem underlines a partir do tamanho da palavra escolhida através da função(len()) da variável(palavra) -> len(palavra). a variável (tentativa) armazena o valor(6) que é o número máximo de tentativa do usuário. A lista(letras_erradas) é criada com nada dentro dela, por que será usada para armazenar as letras que não corresponde as letras da palavra escolhida. E logo à frente imprime("Bem vindo ao jogo da Forca!")

Dentro da função jogar, no bloco seguinte temos uma estrutura de repetição(while), que enquando o números de tentativas for maio 0 e a palavra mascarada for diferente da palavra escolhida(palavra) a estrutura de repetição executarar toda lógica de repetição enquanto o usuário tiver tentativas restantes e não tiver acertado a qual a palavra foi mascarada

Ainda dentro da função jogar temos uma verificação que ocorrerar quando a estrutura de repetição parar, irá verificar se a palavra mascarada é igual a palavra escolhida se sim, Imprimirá ("Parabéns! Você acertou a palavra (palavra que estava oculta))" senão imprime("ocê perdeu! A palavra era (palavra que estava oculta)")

E por fim. Temos um bloco que verifica se o nome do modulo _name_ é igual á "_main_" se sim, chama a função jogar
